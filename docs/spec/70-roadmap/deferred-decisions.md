# Deferred Decisions

## Purpose

This document captures unresolved choices that should not be resolved ad hoc during implementation.

## Deferred Items

- Exact local persistence technology for session and queue state
- Exact initial tool catalog included in MVP
- Exact DeepSeek endpoint and model selection
- Exact compaction size thresholds
- Maintenance-mode semantics for bedrock mutation
- Whether tool exposure per step is full-registry or curated subset in MVP

## Rule

Implementers should not invent durable answers to these questions outside an explicit spec or ADR update.
