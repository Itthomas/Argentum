# Post-MVP Expansion

## Purpose

This document records likely expansion paths that are intentionally excluded from MVP.

## Candidate Expansions

- Maintenance mode for controlled bedrock mutation
- Additional provider adapters beyond DeepSeek
- Additional channel adapters beyond terminal CLI
- Background or asynchronous compaction
- Parallel tool execution
- Stronger sandbox isolation
- Long-term memory tooling and retrieval
- Multi-agent orchestration

## Rule

These items are not implicitly in scope for MVP implementation.
